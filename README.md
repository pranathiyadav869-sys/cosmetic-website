## 🌟 Overview

**Beauty Bliss** is a modern, responsive cosmetic e-commerce website built using HTML, CSS, and JavaScript.
It provides a clean and elegant interface for users to explore beauty products and navigate through popular cosmetic brands.

This project focuses on **front-end development**, UI design, and basic interactivity.

---

## ✨ Features

* 🎨 Elegant and visually appealing UI design
* 📱 Responsive layout for different screen sizes
* 🛍️ Product showcase with images and pricing
* 🔘 Interactive **"Shop Now"** button
* 🌐 Quick navigation to popular cosmetic brands
* ⚡ Lightweight and fast-loading website

## ⚙️ How It Works

### 1. Navigation Bar

Provides links to different sections like Home, Shop, About, and Contact.

### 2. Hero Section

* Displays the main tagline
* Includes a **Shop Now** button
* On click → shows/hides brand options using JavaScript

### 3. Brand Section

Users can directly visit:

* Lakme
* Dior
* Sugar Cosmetics
* Maybelline

### 4. Product Section

Displays product cards:

* Image
* Product name
* Price
* "Add to Cart" button (UI only)

### 5. Footer

Contains copyright information.

## ▶️ How to Run the Project

1. Clone the repository:

```bash

```

3. Run the project:

* Open `index.html` in your browser
